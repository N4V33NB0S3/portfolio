Python Cheat Sheet: A Netrunner's Guide

Here's a quick reference for some of the most common tools in your Python arsenal. Think of it as your personal cyberdeck quick-menu.

Part 1: Core System Functions

These are your basic commands, the bread and butter of any script.

FunctionWhat It DoesExample
print(value)The classic way to show something on screen. Essential for seeing what your code is doing.print("Jack in!")
len(item)Tells you the size of something, like characters in a string or items in a list.len([1, 2, 3]) // returns 3
type(variable)Identifies what kind of data you're dealing with (number, text, list, etc.).type(10) // returns <class 'int'>
int(value)Forces a value to become a whole number. Good for converting text digits to actual numbers.int("5") // returns 5
str(value)Turns any value into a string (text).str(123) // returns '123'
list(item)Converts something iterable (like text) into a list of its parts.list("abc") // returns ['a', 'b', 'c']
sum(list)Quickly adds up all the numbers in a list.sum([10, 20, 5]) // returns 35
sorted(list)Gives you a new, sorted version of a list without messing up the original.sorted([3, 1, 2]) // returns [1, 2, 3]

Part 2: String Manipulation

Text is data. Here's how you slice it, dice it, and put it back together.

MethodWhat It DoesExample
.upper()SHOUTS YOUR TEXT. Converts to uppercase.'hello'.upper() // returns 'HELLO'
.lower()Whispers your text. Converts to lowercase.'HELLO'.lower() // returns 'hello'
.strip()Cleans up messy text by removing whitespace from the ends.' hi '.strip() // returns 'hi'
.replace(old, new)Finds a piece of text and swaps it with something else.'a-b-c'.replace('-', ' ') // returns 'a b c'
.split(separator)Breaks a string into a list of smaller strings. Perfect for parsing data.'192.168.1.1'.split('.') // returns ['192', '168', '1', '1']
.join(list)The opposite of split. Takes a list of strings and joins them into one.'-'.join(['a', 'b']) // returns 'a-b'

Part 3: List Operations

Lists are your go-to for storing collections of items in order.

MethodWhat It DoesExample
.append(item)Adds a single item to the very end of your list.my_list.append(4)
.extend(list2)Merges another list onto the end of your current one.my_list.extend([5, 6])
.pop(index)Removes and gives you an item from a specific position. If you don't say where, it takes the last one.my_list.pop()
.remove(item)Hunts down and removes the first item that matches what you're looking for.my_list.remove(3)
.sort()Sorts the list for real, modifying the original. No going back.my_list.sort()

Part 4: Dictionary Tactics

Dictionaries store data as key-value pairs, like a high-tech filing cabinet.

MethodWhat It DoesExample
.keys()Gives you all the keys (the labels) from your dictionary.my_dict.keys()
.values()Gives you all the values (the data) from your dictionary.my_dict.values()
.items()Hands you key-value pairs together, perfect for looping through.my_dict.items()
.get(key, default)Safely gets a value. If the key doesn't exist, it gives you a default value instead of crashing.my_dict.get('b', 0)
